# Some Yazi commands 💻

## File Operations
C => **for copying**
y,x => **Yanking**
Y,X => **Cancel Yanking**
p,P => **paste**, paste overriding 
d,D => **Trash**, Permanent delete
O,o => for **opening** with editor / interactively 
a => **create a file**
r => Rename

## Navigation
g => **go to** («space» interactively)

; => **Shell** command

## Filtering
f => **filter** 

## Info
tab => **Gives info** about the file/dir

## Tabing 
t => **tab**
ctrl + c => close tab 
1,2,3 => cycle through tabs 
/,? => **Find** next / previous
